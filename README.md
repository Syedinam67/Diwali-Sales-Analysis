# 🪔 Diwali Sales Analysis

A Data Analysis project that explores customer purchasing behavior during the Diwali festival using Python. The project focuses on data cleaning, exploratory data analysis (EDA), and interactive visualizations to uncover sales trends, customer demographics, and product performance. A Power BI dashboard was created to provide actionable business insights.

## 🚀 Project Overview

This project analyzes a real-world Diwali Sales dataset to answer key business questions such as:

- Which states generate the highest sales?
- Which age group contributes the most revenue?
- What are the top-selling product categories?
- How do gender and marital status influence purchasing behavior?
- Which occupations contribute the highest sales?

The project demonstrates the complete data analysis workflow from data preprocessing to visualization.

## 📌 Key Insights

- Uttar Pradesh generated the highest sales revenue.
- Customers aged **26–35** contributed the largest share of purchases.
- Female customers accounted for a higher proportion of sales than male customers.
- Food, Clothing, and Electronics were the top-performing product categories.
- Customers working in the IT sector generated the highest revenue.
